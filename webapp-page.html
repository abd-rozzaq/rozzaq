Web App Design for Intelia Technologies

This software is used by agricultural producers to access information relating to their production. The software allows them to ensure that the breeding conditions are optimal and thus allows them to maximize their yield.

Indirectly, by controlling their production more efficiently, there is a significant impact on the environment. Example: reduction of CO2 emissions. Reduction of ammonia emission. Reduction in the use of antibiotics.

The GlobalView page allows them to view all of their production buildings.

The OneView page allows them to view each building individually. And the ranking page allows them to compare themselves to their colleagues (other farmers).

Client Requirements

They are looking for a clean modern design of our Web based software solution. They currently have a working solution but are looking for a vast improvement of our UI UX.

The User & Audience

They are building a SaaS used for farm management. It as be very clean and useful, the learning curve to use the service should be ZERO, so UX is a prime. Target audience is farmers. There are looking for an easy to use offer.

Role & Responsibility

I act as UI Designer, who designs the interface of this web app.
